# 3.4 工作台模块（采购侧）

## pages/ 已有页面
- `workbench/workbench_home.html`：采购专员工作台首页

- `workbench/workbench_home.html` → `after_sales/after_sales_buyer.html`
- 状态卡片跳转（自动打开对应 tab）：
  - `after_sales/after_sales_buyer.html?tab=pending`
  - `after_sales/after_sales_buyer.html?tab=return`
  - `after_sales/after_sales_buyer.html?tab=shipping`
